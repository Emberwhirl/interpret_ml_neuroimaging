derivatives folder includes:
- trial_images: single trial beta maps for each subject and heat and rejection condition
- contrast_images: contrast images for each subject and heat, warmth, friend, and rejection conditions 
- nuisance.mat Ð example nuisance variable (head motion parameters roll, pitch, yaw, x, y, z) matrix for each subject and trial
- mini_example_social_physical_masked_cross.hdf5.zip - an example of one training set and one test set of trial data in .hdf5 format for training and testing a CNN model
